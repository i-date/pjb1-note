# VBAショートカット


|ショートカット内容|ショートカットキー|
|:-----------------|:-----------------|
| VBE起動 | Alt+F11 |
| プロシージャ実行  | F5 |
| プロシージャ一行ずつ実行  | F8 |
| 候補検索 | Ctrl+Space |
| ヘルプ | 対象選択→F1 |
| オブジェクトブラウザー | F2 |
| イミディエイトウインドウ表示 | Ctrl+G |

- ショートカット作成: コメント化 / 非コメント化
  1. 表示→ツールバー→編集
  2. 編集ツールバー右クリック→ユーザー設定
  3. 選択したボタンの編集→名前変更(&キーで指定)→イメージとテキストを表示
